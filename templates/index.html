What is Flask?

Flask is a microframework for python based on the Werkzeug toolkit and Jinja 2 template engine. Flask is BSD licensed, meaning, developers have the ability to do what they want as long as the conditions are not modified and the disclaimer is present.

Websites such as Pinterest and LinkedIn use the Flask framework. Receiving as many as 12 billion requests for the API alone in Flask everyday

Project Setup

Relatively an easy setup. Imported Flask into the Pycharm development environment which gave us the ability to clone directly from github. Followed implementation directions from the Flask documentation found on the website.

Alternatively, can clone from a terminal to achieve the same result.


Team worked with both Python 2.7 and Python 3.5.2 and operated in both OSX and Windows.
